---
title: How to add tasks via email-in
---
1. **<u>Copy email-in address:</u>** Right-click or click the three-dots icon and select “**Copy email-in address**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-10.14.58-PM.png" width="870" height="752" alt="Copy email-in address in ProofHub" />

2. **<u>Compose email:</u>** Open your email client and create a new email. Paste the copied email-in address into the recipient field.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2025-01-21-at-4.37.05-PM.png" width="581" height="576" alt="Email-in task" />



3. **<u>Use email body format:</u>** For a more organized approach, you can use specific formats in the body of the email to define task details. Choose one of the following formats:\


   ```typescript
   [Task] Task name
   [Desc] Description
   [Start] 01/12/2024
   [End] 01/15/2024
   [Subtask] Subtask name
   [Desc] Subtask description
   [Start] 01/12/2024
   [End] 01/17/2024
   ```

4. **<u>Send email:</u>** Once you’ve entered all the task details in the email, send it to the email-in address associated with your selected tasklist in ProofHub. The content of your email will be processed, and a task will be created in the selected task list based on the provided details.

**Prerequisites:** Before using the email-in feature in ProofHub, ensure the following conditions are met:

- Email id needs to verified.

- You have logged into your ProofHub account at least once.

- You are assigned to the project where you want to add tasks via Email-in.

- The email address used for Email-in is the same as the one associated with your ProofHub account.
