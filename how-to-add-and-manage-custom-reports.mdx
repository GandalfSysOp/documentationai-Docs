## **Introduction to a custom reports**

With ProofHub, you can visualize every aspect of your project and its overall progress in one place. Additionally, you can track an individual’s tasks, milestones, time data, and overall progress. Utilize individual reports to analyze performance and progress across all their projects, and plan ahead by leveraging ProofHub’s resource management reports.

Under the custom report section, you can create the following types of reports:

- Tasks report

- Milestones report

- Time report

## **Tasks report**

The task report helps you keep track of open and overdue tasks, showing who is behind on which tasks and by how many days. This allows you to plan accordingly.

1. **<u>Navigate to the reports section:</u>** Click on the **“+”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/custom-task-report.png" width="329" height="437" alt="" />

2. Select **“Custom report”** and click **“Next”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Task-report-2.png" width="608" height="723" alt="" />

   - Add a **“Title”** and select projects for which you would like to retrieve the tasks report.

   - Under “**Create a”&#x20;**&#x6F;ption: Select **“Tasks report”**.

   - **Assigned to:** Select a person to see the task assigned to a specific resource.

   - **Created by:** Select a person if you want the task created by a specific person.

   - **View tasks:&#x20;**&#x53;elect the multiple options available under the “View tasks” filter, to see all, overdue, due, completed, and the tasks for which you haven’t set any date.

   - **Labels:** Select labels if you want to retrieve the task report for specific labels.

   - **Group by:** Group by using the following options:

     - Date

     - Project

     - Tasklist

   - Enable the **“Include details”** option to see all the details and click **“Add”**.

3. A report will be generated which you can download in PDF format.

## **Milestones report**

To view overdue, completed, and upcoming milestones, you can visualize their progress using straightforward roadmap timelines and charts.

1. **<u>Navigate to the reports section:</u>** Click on the **“+”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/custom-milestone-report.png" width="329" height="437" alt="" />

2. Select **“Custom report”** and click **“Next”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Milestone-1.1.png" width="686" height="717" alt="" />

   - Add a **“Title”** and select projects for which you would like to retrieve the milestones report.

   - Under the **“Create a”&#x20;**&#x6F;ption: Select **“Milestones report”**.

   - **Assigned to:** Select a person to see the milestone assigned to a specific person.

   - **Created by:** Select a person if you want the milestone created by a specific person.

   - **View milestones:&#x20;**&#x53;elect the multiple options available under the **“View milestones”** filter, to see all, overdue, due, and completed milestones.

   - **Group by:** Group by using the following options:

     - Date

     - Project

   - Enable the **“Include details”** option to see all the details and click **“Add”**.

3. A report will be generated which you can download in PDF format.

## **Time report**

This report provides a clear overview of the time logged by individuals for billable, billed, and non-billable tasks.

1. **<u>Navigate to the reports section:</u>** Click on the **“+”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/custom-time-report.png" width="329" height="437" alt="Navigate to the report section for custom time report in ProofHub" />

2. Select **“Custom report”** and click **“Next”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/time-report-1.png" width="683" height="693" alt="" />

   - Add a **“Title”** and select projects for which you would like to retrieve the time report.

   - Under the “**Create a”&#x20;**&#x6F;ption: Select **“Time report”**.

   - **Status:** Select the status from the billed, billable, non-billable, void, and none.

   - **Logged by:** Select a person if you want to see the time entries made by a specific person.

   - **Logged:&#x20;**&#x53;elect the multiple options available in the “Logged” option to see the time logged as per the dates.

   - **Group by:** Group by using the following options:

     - Project

     - Timesheet

     - Person

     - Date

   - Enable the **“Include details”** option to see all the details and click **“Add”**.

3. A report will be generated which you can download in PDF format.

## **Custom field report**

Custom field reports enable you to track and analyze custom fields across multiple projects or task lists within a single report.

1. **<u>Navigate to the reports section:</u>** Click on the **“+”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/custom-field-report-1-1.png" width="329" height="437" alt="" />

2. Select **“Custom field report”** and click **“Next”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/custome-field-1.png" width="679" height="735" alt="" />

3. **<u>Enter the report details:</u>**\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Custom-field-2.png" width="685" height="385" alt="" />

   \


   - Add a **“Title”** and select projects or tasklist for which you would like to retrieve the custom field report.

   - **X-axis:** Create a report to analyze the custom field and task count relation. The X-axis will reflect the custom fields **“Multiselect”** and **“Dropdown”** and also it will show the project, task workflow stage, and priority.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Custom-field-3.png" width="683" height="422" alt="" />

   - **Y-axis:** The y-axis will reflect the custom fields **“Currency”**, **“Numbers”**, and **“Percentage”** and also you can see the task count.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Custom-field-4.png" width="737" height="463" alt="" />

   - **Filters:** Select the filter as per your requirements from the options: **“Completion status”**, **“Completion date”**, **“Creation date”** and **“Due date”**.

   - Enable the “**Include subtasks”** option to see all the details.

   - **Mark as private:** Check the “**Mark as private**” option if you need to restrict the visibility of the report and click **“Add”**.

4. A report will be generated which can be viewed in both graphical and tabular forms.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Custom-field-report-graph.png" width="996" height="528" alt="Custom field report graph in ProofHub" />

**<u>Note:</u>** *You can download the custom field report in PDF or CSV format.*
