## **All project report**

With the **“All project report”** feature, you can retrieve a comprehensive report of all the projects within the account. It is also a default report and the default report cannot be altered.

1. **<u>Navigate to the reports section:</u>** Under **“Work report”** select **“All projects report”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/All-project-report-1.png" width="515" height="435" alt="" />

2. A report will be generated where you will be able to see the following information related to all projects within your account:\


   - **Summarized information:** This will show you the total number of projects, archived projects, and project distribution.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-1.png" width="982" height="98" alt="" />

   - **Project status:** You can view the number of projects under different statuses.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-2.png" width="1004" height="249" alt="" />

   - **Project category:** You can view the number of projects under different project categories.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-3.png" width="1003" height="357" alt="" />

   - **Project information:&#x20;**&#x59;ou will be able to view all the names of the projects, their manager, start and due dates, assignees, and task completion percentages.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-4.png" width="1000" height="359" alt="" />



**<u>Note:</u>** *You can download all project reports in PDF format.*

## **Project report**

You can retrieve summarized details specific to an individual project using the project report feature. It is a default report and the default report cannot be altered.

1. **<u>Navigate to the reports section:</u>** Under **“Work report”** select **“Project report”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/project-report-1.png" width="733" height="434" alt="" />

2. In the default report for the selected project, you’ll see the following information:

   - **Summarized information:** You will be able to see the summarized information of the selected project such as project manager, project status percentage, open tasks versus total tasks, etc.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-5.png" width="1013" height="323" alt="" />

   - **Roadmap – Milestones**: You’ll be able to see a timeline for the entire project, with start and end dates marked, and all milestones will appear on the dates they are due. It will also show you how much the project is completed.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-6.png" width="977" height="432" alt="" />

     \
     **<u>Note</u>**<u>:</u> *The overdue, upcoming, and completed milestones are denoted with separate colors of red, black, and green respectively. You can see a table containing the list of all milestones in chronological order.&#xA0;*

   - **Burn up chart**: You’ll be able to see two line graphs that track completed work/tasks and total work/tasks. It’ll help you to know if the project is not yet completed because work is being done slowly, or if too much work is being added.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-7-1.png" width="1087" height="366" alt="" />

   - **Overdue tasks**: A list of all overdue tasks present in the project will show up here. These overdue tasks are shown in tabular form and arranged in chronological order.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-8.png" width="1019" height="149" alt="" />

   - **Tasks by labels**: You can see the count of tasks on which a label is assigned that are overdue/open, completed, and total. Also, you’ll see the count of total tasks to which the label is assigned.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-9.png" width="1035" height="398" alt="" />

   - **Tasks by resource**: You’ll be able to see the count of tasks on which a person is assigned that are overdue/open completed and the total. Also, you’ll see the count of total tasks to which a person is assigned.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-10.png" width="1033" height="341" alt="" />

   - **Task by Workflow**: You’ll see a stacked bar graph for each workflow in which you’ll see open, overdue, and completed tasks. For each workflow, you’ll see information in tabular form and the bar chart form.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-12.png" width="1282" height="475" alt="" />

     \
     In the tabular form, you’ll be able to see the following information:

     - Stage name

     - Count of overdue tasks

     - Count of the total task\


       <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-13.png" width="1211" height="409" alt="" />

   - **Time logged**: Information about the time logged in a project is available here. The logged time is displayed by status, with a table showing the status name and the total time for each status. Additionally, a pie chart is provided to visualize this data.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-14.png" width="1282" height="399" alt="" />

     \
     Information on time logged by resources and timesheets is displayed in a table, which includes the resource name, status name, and total time logged.\


     <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-15.png" width="1151" height="425" alt="" />

**<u>Note:</u>** *You can download the project report in PDF format*.

## **Project status report**

A project status report allows you to view the status of a project category/project. You can view the status of all projects belonging to the selected projects or project categories.

1. **<u>Navigate to the reports section:</u>** Click on the **“+”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/project-status-report-1.png" width="329" height="437" alt="" />

2. Under **“Work report”** select **“Project status report”** and click **“Next”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/project-status-report-2.png" width="762" height="736" alt="" />

3. **<u>Enter the report details:</u>**\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/project-status-3.png" width="758" height="425" alt="" />

   - Add **“Title”** and select the project or project categories for which you would like to create a project status report.

   - **Group by:** Group by using any of the following options:

     - Project

     - Project category

     - Project statuses

     - Project manager

   - **Filters:** Add multiple filters as per your requirements such as project status, project manager, project start and due date, project creator, and project created date.

   - **Mark as private:** Check the “**Mark as private**” option if you need to restrict the visibility of the project status report.

4. A report will be generated which can be viewed in either graphical or tabular format, based on your preference.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/all-project-16.png" width="1010" height="450" alt="" />

**<u>Note:</u>** *You can download the project status report in PDF or CSV format.*
