---
## **Get Categories**

- `GET v3/categories` will return all account Categories.

```typescript
[
	{
		"id":65707070,
		"name":"Marketing Projects",
		"default":false
	},
	{
		"id":65707071,
		"name":"Uncategorized",
		"default":true
	}
]
```

## **Get Category**

- `GET v3/categories/65707070` will return the specified Category.

```typescript
{
	"id":65707070,
	"name":"Marketing Projects",
	"default":false
}
```

## **Create Category**

- `POST v3/categories` will create a new Category.

```typescript
{
	"name": "Rockwall Phase I"
}
```

`201 Created` will be returned along with the JSON of the category (<a href="https://github.com/ProofHub/api_v3/blob/master/sections/categories.md#get-category" target="_blank" rel="noopener noreferrer"><u>Get Category</u></a>) if the record is added.

## **Update Category**

- `PUT v3/categories/65707070` will update the category.

```typescript
{
	"name": "Rockwall Phase II"
}
```

`200 OK` will be returned along with the JSON of the category (<a href="https://github.com/ProofHub/api_v3/blob/master/sections/categories.md#get-category" target="_blank" rel="noopener noreferrer"><u>Get Category</u></a>) if the record is updated. `403 Forbidden` will be returned in case of invalid access.

## **Delete Category**

- `DELETE v3/categories/65707070` will delete the category specified.

`204 No Content` will be returned if the record is deleted. `403 Forbidden` will be returned in case of invalid access.
