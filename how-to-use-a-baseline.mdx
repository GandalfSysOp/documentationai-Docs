---
title: How to use a baseline
---
A baseline is a fixed reference point that represents the original project plan. It includes the initially planned start and finish dates for each task, and serves as a standard against which actual progress and any deviations can be measured.\
This will help the project managers to track the progress of tasks. It enables the project manager to assess, at a glance, whether the project is on track and how far the project has deviated from the initial baseline set.

## **Configure baseline**



<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Baseline1.png" width="1356" height="414" alt="" />

1. Click on the three dots at the top right of the page and select **“Set baseline”**.

2. After successfully setting the baseline, ensure to select the **“Show Baseline”** option.

## **Enhance project planning with baseline feature**

1. Create your plan as per the schedule and set the baseline dates. \


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Baseline-2.png" width="1353" height="387" alt="" />

2. Beneath the task line, a grey line representing the baseline will be visible. Hovering over it unveils the baseline start and end dates. Similarly, hovering over the task line reveals the current start and end dates reflecting the task’s progression. This comparison aids project managers in strategizing current progress and effectively managing the work schedule.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-23-at-3.24.10-PM.png" width="1032" height="511" alt="" />

   \


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-23-at-3.19.07-PM.png" width="1034" height="518" alt="" />

3. Once the baseline is set for the tasks, all the baseline dates get frozen and can be viewed in the baseline columns at the left.   \


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Baseline-5.png" width="1290" height="350" alt="" />

**<u>Note:</u>***&#x20;Only a project manager can set the baseline for a project.*
