---
title: How to manage file versioning
---
In ProofHub, a file version represents each distinct iteration or update of a file that has been uploaded. This allows users to track changes, revert to previous versions if necessary, and maintain a history of revisions. File versioning in ProofHub helps streamline collaboration by ensuring that team members can access the most recent version of a file and track its evolution over time.

## **Upload a new version of the file**

1. Right-click on the file or click on the three dots and select the **“Upload new version”&#xA0;**&#x6F;ption.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Upload-a-new-version-of-file.png" width="1356" height="475" alt="Upload a new version of file in ProofHub" />



2. Upload the new version of the file, select people to be notified, click **“Ok,”** and the new version will be uploaded.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/81.png" width="707" height="341" alt="" />

---

## **View version history**

1. Go to the top right of the screen click on the **“i”** icon and select the **“Version history”** option.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Version-history.png" width="1310" height="494" alt="Version history in ProofHub's files section" />



2. Or you can view the version history of the file by right-clicking on the file or clicking on the three dots and selecting the **“Version history”&#xA0;**&#x6F;ption.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Version-history-by-clicking-on-3-dots.png" width="1359" height="465" alt="Version history by clicking on 3 dots in ProofHub" />

   \
   \
   **<u>Note:</u>** *The current version of the file will appear with a current label.&#xA0;*

---

**Convert the previous version to the current**\
\



<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Make-previous-version-current.png" width="1361" height="553" alt="Convert the previous version to the current in ProofHub's files section" />

1. Right-click on the file and select **“Version history”**.

2. A window will open on the right from where you can view all the versions of the file. 

3. Click on the “**Make current”** option to convert a previous version into the current version of the file.

**<u>Note:</u>****&#xA0;***Once you make the selected version as current, all versions uploaded after it will be deleted.*

---

## **Delete the previous version**



<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Delete-the-previous-version.png" width="1358" height="551" alt="Delete the previous version in ProofHub's files section" />

1. Right-click on the file and select **“Version history”**.

2. Hover over the previous version that you wish to delete and click on the three dots.

3. Select “**Delete”** from the context menu.

4. A confirmation window will appear on the screen. Click **“Yes”&#xA0;**&#x74;o confirm the deletion.

**<u>Note:</u>** *The previous version deleted can also be restored from&#xA0;****“Trash”****. The restored file will appear like the previous version on the same file.*

---

## **Download the previous version**

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/06/Download-the-previous-version-1024x416.png" width="1024" height="416" alt="" />

1. Right-click on the file and select **“Version history”.**

2. Hover on the previous version that you wish to download and click on the three dots icon.

3. Select **“Download”&#xA0;**&#x66;rom the context menu.
