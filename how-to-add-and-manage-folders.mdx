## **Folder hierarchy**

In ProofHub, you have the ability to organize your uploaded files into a hierarchy of folders.\
\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Folder-hierarchy.png" width="1363" height="555" alt="Folder hierarchy in ProofHub's Files section" />

1. **<u>Project files</u>****:** By default, the **“Project files”** folder is displayed, and any additional folders you create are nested within this main folder. You have the flexibility to create folders within folders to any desired depth.

2. **<u>Tree-like structure</u>****:** The folder arrangement follows a tree structure, with the **“Project files”** folder serving as the base or root. To access files within a specific folder, simply click on the folder name.

3. **<u>All files</u>****:** If you wish to view all files irrespective of the folder they have been added into, you can do so using the **“All files”**.

4. **<u>Search and add folders:</u>** You can use the magnifying glass icon to search for folders and subfolders. Clicking on the **“+”** icon allows you to add folders and subfolders.

## **Add a folder**

1. **<u>Navigate to the files section</u>****:** Click on the **“Add”** button and select **“Folder”** from the drop-down.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Add-folder-in-Files.png" width="1324" height="432" alt="Add folder in ProofHub's files section" />



2. **<u>Enter the folder details</u>****:&#xA0;**&#x45;nter the title and click **“Add”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/59.png" width="703" height="202" alt="" />

## **Add a sub-folder&#xA0;**

Right-click or click on the three dots in front of the folder name where you intend to add the subfolder and select the **“Add sub folder”** option.\
\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Add-subfolder-in-Files.png" width="1308" height="558" alt="Add subfolder in ProofHub's files section" />

---

## **Download a folder**

Right-click or click on the three dots in front of the folder name and select the **“Download”** option.\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Download-files-in-folders.png" width="1306" height="550" alt="Download files in folders in ProofHub's files section" />

**<u>Note:</u>** *All the files present in that folder will be downloaded in a zip folder.*

---

## **Move a folder**

1. Right-click or click on the three dots in front of the folder name and select the **“Move”&#xA0;**&#x6F;ption.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Move-files.png" width="1307" height="563" alt="Move files to another folder in ProofHub" />



2. Select the folder where you want to move the files and click on **“Update”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Files-2.png" width="486" height="184" alt="" />

---

## **Rename a folder**

1. Right-click or click on the three dots in front of the folder name and select the **“Rename”** option.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Rename-folder.png" width="1265" height="561" alt="Rename folder in ProofHub's files section" />

2. Enter the new name and click on **“Update”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/1-1.png" width="698" height="197" alt="" />

---

## **Delete a folder**

1. Right-click or click on the three dots in front of the folder name and select the **“Delete”&#xA0;**&#x6F;ption.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Delete-folder-in-Files-section.png" width="1286" height="560" alt="Delete folder in ProofHub's files section" />



2. When deleting a folder, all of its subfolders will be removed, but the files remain accessible in **“All files”.** If you want to completely remove the files, ensure to select the **“Remove files from ‘All files’ too”** option and confirm by clicking **“Yes”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/66.png" width="488" height="266" alt="" />

***<u>Note:</u>******&#xA0;****ProofHub keeps items in the “Trash” for 15 days. Restore them easily by going to Trash. For detailed instructions on restoring deleted items from the trash, please refer to&#xA0;*<a href="https://help.proofhub.com/swift/manage/how-to-manage-trash" target="_blank" rel="noopener noreferrer">*Trash*</a>*.*
