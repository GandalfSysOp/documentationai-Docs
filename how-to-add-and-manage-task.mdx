## **Add a task**

1. **<u>Open the project and task list:</u>** Click on the “**Add**” button at the top of the screen to open the task creation window.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.07.35-PM.png" width="1440" height="751" alt="" />

2. **<u>Enter task details:</u>** In the task creation window, fill in the following details:


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-11.04.02-AM.png" width="715" height="574" alt="" />

   - **Title:** Enter a title that clearly describes the task.

   - **Description:** Provide a detailed description of the task, including any necessary information or context.

   - **Fields:**

     - **Start date and due date:** Set the task’s start and end dates.

     - **Labels:** Assign relevant labels to categorize and identify the task.

     - **Assignees:** Select team members responsible for completing the task.

     - **Estimated time:** Specify the estimated time required to complete the task.

     - **Custom fields:** If applicable, add custom fields to capture specific information related to the task. Refer to additional documentation for details on <a href="https://help.proofhub.com/swift/tasks/how-to-add-and-manage-custom-fields" target="_blank" rel="noopener noreferrer">custom fields</a>.

   - **Add files & documents:** Attach any necessary files or documents related to the task.

3. **<u>Save the task:</u>** Click on the “**Add**” button.

## **Add a subtask**

1. **<u>Add a new subtask:</u>** While using the table view, under the “**Subtask**” column next to the task you want to add subtasks. Move your cursor within the Subtask column of the desired task and click on the “**+**” icon. Enter the title of the subtask and press “**Enter**” on your keyboard to create the subtask.

## **Manage people in task**

1. **<u>Access assignees</u>**: Right-click on the task name and select “**Manage people**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.12.53-PM.png" width="1440" height="753" alt="" />

2. **<u>Add or remove assignees:</u>** Click on the “**Manage People**” option to see a list of people assigned to this task.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-4.12.34-PM.png" width="397" height="193" alt="" />

3. **<u>Select assignees:</u>** In the assignees’ window, you’ll see a list of existing assignees who have access to this task.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-4.13.45-PM.png" width="396" height="430" alt="" />

4. Tick the checkbox next to the names of the people you want to add as assignees or uncheck to remove them as assignees, and click on the “**Update**” button.

## **Delete a task**

1. **<u>Select the task</u>**: Right-click on the task name and select “**Delete**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.13.48-PM.png" width="1440" height="753" alt="" />

2. **<u>Confirm deletion</u>**: Select “**Yes**” within this confirmation window to proceed with deleting the task.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-4.18.58-PM.png" width="447" height="173" alt="" />

**<u>Note:</u>** *ProofHub keeps items in the “Trash” for 15 days. Restore them easily by going to Trash. For detailed instructions on restoring deleted items from the trash, please refer to&#xA0;*<a href="https://help.proofhub.com/swift/manage/how-to-manage-trash" target="_blank" rel="noopener noreferrer">*Trash*</a>*.*

## **Create a copy of a task**

1. **<u>Select the task:</u>** Right-click on the task name and select “**Create a copy**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.14.41-PM.png" width="1440" height="751" alt="" />

2. In the “**Copy task**” window that opens:


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-4.48.26-PM.png" width="718" height="567" alt="" />

   - **Title:** Change the task title for the new copy if needed.

   - **Project:** Choose the project where the new task copy will be created.

   - **Tasklist:** Specify the task list within the selected project where the task copy will be added.

   - **Select the stage of the task:** Select the task stage (e.g., To-Do, In Progress, Completed) within the task list where the task copy will be placed.

   - **<u>Copy from the original task:</u>**

     - **Assignees:** Select this option to copy the same assignees from the original task to the new task copy.

     - **Dates:** Check this box if you want to copy the start date and due date from the original task to the new task copy.

     - **Comments:** Enable this option if you wish to copy comments associated with the original task to the new task copy.

## **Share task**

1. **<u>Select the task:</u>** Right-click on the task name and select “**Share**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.15.37-PM.png" width="1440" height="751" alt="" />

2. **<u>Select a sharing option:</u>** In the “**Share**” menu, you will find two sharing options:


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-4.58.24-PM.png" width="342" height="261" alt="" />

   - **Copy task link:** Select this option to copy the URL path of the task to your clipboard. You can then share this link with others via any communication platform.

   - **Share via Email:** Choose this option to share the task directly via email.

---

## **Move a task**

1. **<u>Select the task:</u>** Right-click on the task name and select “**Move**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.16.25-PM.png" width="1440" height="751" alt="" />

2. **<u>Customize details:</u>** In the “**Move task(s)**” window that opens:


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-15-at-5.06.58-PM.png" width="717" height="530" alt="" />

   - **Title:** Change the task title if needed.

   - **Project:** Select the project where the new task will be moved.

   - **Tasklist:** Specify the task list within the selected project where the task will be moved.

   - **Move from the original task:**

     - **Assignees:** Select this option to keep the same assignees.

     - **Dates:** Check this box if you want to keep the start date and due date from the original task.

     - **Comments:** Enable this option if you wish to keep comments associated with the original task.

     - **Custom fields:** Any custom fields created will be moved as well.

**<u>Note:</u>** *If a task gets moved to a different project or list, its URL will change. This means the old link won’t work anymore.*

## **Searching tasks by Task ID**

1. You can easily search for a specific task using its Task ID from the advanced search option in ProofHub.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-3.51.29-PM.png" width="468" height="754" alt="" />

2. To search for a task by Task ID, input the Task ID prefixed with # (e.g., “#1234”) into the advanced search.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-23-at-9.45.33-AM.png" width="717" height="236" alt="" />

**Note:** *For detailed instructions about advanced search, please refer to&#xA0;*<a href="https://help.proofhub.com/swift/others/introduction-to-search" target="_blank" rel="noopener noreferrer">*Advanced search*</a>*.*

## **Show or hide Task IDs**

1. People can show or hide Task IDs at the task level within ProofHub.

2. Click on the three-dots icon at the top and select “**Show columns**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.17.24-PM.png" width="1440" height="753" alt="" />

3. Under “**Show/hide columns**“, check or uncheck “**Task ID**” based on your preference.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.18.38-PM.png" width="1440" height="753" alt="" />
