---
title: How to view task dependency
---
Task predecessors and successors in ProofHub allow you to establish dependencies between tasks, facilitating efficient project management and workflow coordination.

**<u>Visualizing predecessor and successor:</u>**

- Click on the icon within the task window to visualize the predecessor and successor task view.

- This feature provides a graphical representation of task dependencies, enhancing understanding and coordination among project tasks.\

**<u>Note:</u>** *Dependency can be set from Gantt only.*

**<u>Email notifications for task completion:</u>**

- When a task is completed (predecessor task), subscribers assigned to the successor task will receive email notifications.

- This notification informs team members that the successor task can now proceed, ensuring timely task progression within the project workflow.
