title: What is a template
---
Template functionality in project management systems eliminates the need to create projects from scratch repeatedly. Instead, for projects sharing similar structures, templates offer a pre-defined framework. You can select a template during project creation to streamline the process and save time.

## **Add a template**

1. **<u>Access the “Quick add</u>****&#xA0;menu:** Click on the **“+”** icon located in the top left side to open the quick add menu and select **“Template”**.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/4-2.png" width="444" height="487" alt="" />

**<u>Enter the template details</u>****:**

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/5-2.png" width="702" height="556" alt="" />



- **Color:&#xA0;**&#x42;y setting a color for your template, you can utilize the color filter to conveniently sort and organize your templates based on color categories.

- Add th&#x65;**&#xA0;title&#xA0;**&#x61;nd th&#x65;**&#xA0;description.**

- **Assignees:&#xA0;**&#x50;eople who should be part of the project.

- **Project manager:&#xA0;**&#x53;elect the project manager from the list of assignees assigned to the project. The project Manager is responsible for coordinating all aspects of a project, including planning, execution, and team collaboration, to ensure successful completion within defined timelines and objectives.

- Select the **start date** and **end date** of the template.

- Click **Next.**

**<u>Select the project tabs</u>**:

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/6-2.png" width="709" height="566" alt="" />



- **Enable/disable the tabs:** Project tabs refer to the various sections or areas within a project.

- **Set a tab as default:** This means that when you open a project, it will automatically take you to the tab you’ve selected as the default.

1. **<u>Save the template</u>****\*\*\*\*:&#xA0;**&#x41;fter entering all the details and selecting the tabs required for the template, click the **“Add”** button to save your template.

## **How to use a template**

While creating the project check the “**Use template”** option. From the drop-down list select the template you would like to use.

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/7-1.png" width="712" height="649" alt="" />

**<u>Note:</u>** *While creating a project from the template, dates can be adjusted as per the following parameters:*

- *Earliest date in the project/template*

- *Keep the date as it is*

***For example:****\*\* Select the&#xA0;****“Earliest date in the project/template****\*\*” from the drop-down list and set the due date as per your need. Now, if the current earliest due date is December 1 and you move it to a new date, let’s say, December 3, 2 days will be added to all the dates.*

## **Convert the existing project to a template**

1. **<u>Hover over the project name:</u>** The gear icon will be visible, click on it, and select **“Edit”** from the context menu.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Template.png" width="578" height="452" alt="" />

2. Check the **“Is template”** and click on **“Update”** to convert the existing project to a template.
