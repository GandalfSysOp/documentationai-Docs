## **Introduction**

To automate tasks, integrate with other applications, or build custom solutions, you can use the ProofHub API. This guide provides essential information on how to get started with the ProofHub API, including how to generate and use your API key.

## **Getting your API key**

1. **<u>Access profile settings</u>**: Click on the profile icon located at the bottom left corner of the ProofHub interface and select the “**API access**” option.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-07-01-at-3.50.00-PM.png" width="400" height="570" alt="API Access in ProofHub" />

2. **<u>Copy your existing key</u>**: A pop-up window will appear displaying your current API key. You can copy this key directly for immediate use.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-22-at-1.07.22-PM.png" width="716" height="372" alt="" />

3. **<u>Generate a new API key</u>**: If you need to generate a new API key, click on the “**Reset API**” button within the same pop-up window. This will create a new API key for you.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-22-at-1.07.58-PM.png" width="714" height="371" alt="" />

**<u>Note:</u>** *When you reset your API key, any applications using the previous key will be disconnected. You will need to update these applications with the new API key to restore their functionality.*

For detailed information and additional capabilities of the ProofHub API, refer to the official <a href="https://github.com/ProofHub/api_v3" target="_blank" rel="noopener noreferrer">ProofHub API v3 documentation on GitHub</a>.
