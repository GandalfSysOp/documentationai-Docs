## **Create person**

- `POST v3/people` will add a new person to the account.

```typescript
{
    "first_name":"Stella",
    "last_name":"Altois",
    "email":"stella.altois@email.com",
    "role":"90391275",
    "language":"en"
}
```

Language example `en`, `fr`. You can get supported languages from the <a href="https://github.com/ProofHub/api_v3/blob/master/sections/languages.md" target="_blank" rel="noopener noreferrer"><u>languages API</u></a>. You can get the group from the <a href="https://github.com/ProofHub/api_v3/blob/master/sections/groups.md" target="_blank" rel="noopener noreferrer"><u>groups API</u></a>.

## **Update person**

- `PUT v3/people/11679192` will update the person.

```typescript
{
    "first_name":"Stella",
    "last_name":"Altois",
    "email":"stella.altois@email.com",
    "role":"90391275",
    "language":"en"
}
```

`200 OK` will be returned along with the JSON of the person (<a href="https://github.com/ProofHub/api_v3/blob/master/sections/people.md#get-person" target="_blank" rel="noopener noreferrer"><u>Get person</u></a>) if the record is updated. You can get supported languages from the <a href="https://github.com/ProofHub/api_v3/blob/master/sections/languages.md" target="_blank" rel="noopener noreferrer"><u>languages API</u></a>. `403 Forbidden` will be returned in case of invalid access.

## **Delete person**

- `DELETE v3/people/11679192` will delete the person specified.

`204 No Content` will be returned if the record is deleted. `403 Forbidden` will be returned in case of invalid access.
