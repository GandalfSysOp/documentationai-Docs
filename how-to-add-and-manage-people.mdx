## **Add people**

1. **<u>Navigate</u>**<u> </u>**<u>to the people section</u>****:** Find and click the **“Add”** button.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/add-person-1.png" width="545" height="463" alt="" />

2. **<u>Fill in user details</u>****:** A new window will appear for adding user information.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Add-person-details.png" width="703" height="423" alt="" />

   - **Enter first and last name:** Enter the user’s first and last name in the designated fields.

   - **Email address:** Provide a valid email address that the user will use to log in to ProofHub.

   - **Title:** Enter the user’s title (e.g., Marketing Manager)

   - **Access role:** Assign an appropriate access role within ProofHub that defines the user’s permissions.

   - **Select group:** Select a group (or multiple groups) to add the user to for easier organization.

   - **Projects:** Assign the user to specific projects they’ll be working on within ProofHub and click **“Next”**.

3. **Fill further details:**\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Add-person-details-2.png" width="703" height="421" alt="" />

   - **Login setup:** Choose whether to set a password for the user or allow them to create their own upon first login.

   - **User preferences:** Select the user’s preferred language and time zone.

   - **Profile pic:** Upload a profile picture for better identification.

   - **Welcome message:** Craft a personalized welcome message that will be sent to the user upon adding them.

4. **<u>Add User</u>****:** Click the **“Add”** button to save the new user profile.

## **Import people**

1. Find and click the drop-down menu right next to the **“Add”** button.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/import_csv.png" width="543" height="348" alt="" />

2. **<u>Select import CSV</u>****:** Choose **“Import CSV”** from the available options.

3. **<u>Browse and select CSV file</u>****:** A window will pop up, allowing you to browse and select the CSV file containing your user data from your system.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Import-person-csv-2.png" width="708" height="483" alt="" />

4. **<u>Import users</u>****:** Click the **“Import”** button, and all users listed in the properly formatted CSV file will be added to your ProofHub account.\
   \
   **Note:** *A sample CSV is provided to help you understand the order and headings of the columns that should be there in order to successfully import to ProofHub.*

## **Edit people**

1. Click on the three dots icon or right-click on the user name and select “**Edit**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/edit_people.png" width="1371" height="505" alt="" />

2. **<u>Update user information</u>****:** A window will pop up displaying the user’s current details. You can make any necessary changes to their information, such as name, email address, access role, group memberships, project assignments, and other preferences.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/edit-person-details.png" width="705" height="487" alt="" />

3. **<u>Save changes</u>****:** Click the **“Update”** button to save the changes to the user’s profile.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Edit-person-3.png" width="707" height="288" alt="" />

## **Suspend people**

1. Click on the three dots icon or right-click on the user name and select “**Suspend**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/suspend_people.png" width="1371" height="505" alt="" />

2. **Suspend user:** By selecting **“Yes”&#x20;**&#x77;ith the confirmation window, you’ll be temporarily restricting the user’s access to ProofHub. They won’t be able to log in or access any projects or groups until their access is restored.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Suspeneded-people-2-1.png" width="438" height="168" alt="" />

   \
   \
   **Note:** *You can select multiple users at once by checking the boxes next to their names. Then, the “Suspend” option will appear at the top of the list, allowing you to suspend them all simultaneously.*

## **Restoring people**

1. **Filter suspended users:** Click the filter icon located at the top-right corner of the list. This will display all the filter options.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/filer-suspeneded-1.png" width="1316" height="337" alt="" />

2. Select the **“Status”** option from the filter menu. Now, select **“Suspended”&#x20;**&#x6F;ption and click **“Apply”**\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/filter-suspeneded-2.png" width="1314" height="339" alt="" />

3. Find the user you want to restore. Right-click on their name and select **“Restore”** from the context menu.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/restore-suspended.png" width="1318" height="311" alt="" />

4. **<u>User access restored</u>****:** Selecting **“Yes”&#x20;**&#x77;ill grant the user access to ProofHub again. They will be able to log in and resume their work within the platform.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/restore-suspened-2.png" width="441" height="167" alt="" />

## **Sort and filter people**

### **Sorting users**

ProofHub offers two ways to sort your user list:\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/sort-people.png" width="1317" height="330" alt="" />

- **Sort by role:** This option arranges users based on their assigned roles within ProofHub (e.g., admin, normal user, or custom roles). Sorting by role can be helpful when you need to quickly identify users with specific permissions.

- **Sort by name:** This option arranges users alphabetically. Sorting by name is a simple way to find users you know or locate them quickly within a large list.



### **Filtering users**

ProofHub also lets you filter your user list based on user’s role and their status.\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Filter-people.png" width="1298" height="333" alt="" />

- **Filter by role:** You can filter the list to display only users with a particular role (e.g., admin, normal user, or custom roles). This is useful when you need to focus on a specific user group for communication or task assignments.

- **Filter by status:** You can filter based on user status (e.g., active, suspended). This allows you to quickly identify active users or manage suspended accounts.

## **Download people list**

ProofHub offers a convenient way to export a complete list of all users in your account as a CSV file. This downloadable file contains valuable details about your team members, including their names, email addresses, ProofHub status (active, suspended, etc.), and assigned access roles.

You can download this CSV file through the context menu in the People section.\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/download-csv.png" width="1360" height="394" alt="" />

## **Resend invite**

In ProofHub, account owners can send invitations to users after importing projects or to those who have never logged in. The users will receive an invitation containing their login credentials.

- **Send an invite to a** **single person:** Click on the three dots icon or right-click on the user name and select “**Resend invite**” from the context menu to send the invite.\


  <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Resend_invites.png" width="1375" height="616" alt="" />

- **Send invite to multiple people:** Select the users by clicking the checkboxes in front of the name and select **“Resend invite”** to send invites.\
  \
  **<u>Note:</u>** *Account owners can send invitations to people after importing projects from Basecamp, as well as to users who have never logged in before.*
