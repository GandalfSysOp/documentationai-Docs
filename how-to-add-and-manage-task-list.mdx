## **Add a task list**

1. **<u>Navigate to the “Tasks” section:</u>** Select the specific project where you want to create the task list. Click the “**Tasks**” tab on the left-side menu and click on the “**+**” icon to initiate the creation of a new task list.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl1.png" width="360" height="753" alt="" />

2. **<u>Fill task list details:</u>**\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl2.png" width="717" height="530" alt="" />

   - **Title:** Enter a title for the task list.

   - **Associate workflow:** Choose a workflow from the existing dropdown list to define task stages and progress. (For more on workflows, click <a href="https://help.proofhub.com/swift/manage/how-to-manage-workflow" target="_blank" rel="noopener noreferrer">here</a>).

   - **Subscribers:** Add team members who will be involved in this task list.

   - **Subscriber stages:** Select the stage at which subscribers will receive notifications.

   - **Enable time tracking:** Choose a timesheet for logging time spent on tasks within this list.

   - **Associate a milestone:** This milestone could represent an important event or goal related to the task list.

   - **Display list in Gantt:** If you prefer to have a timeline view and manage your task list from the Gantt, select the “**Display list in Gantt**” option.

   - **Mark as private:** Check the “**Mark as private**” option if you need to restrict the visibility of the task list to selected subscribers only.

3. Click on the “**Add**” button.

---

## **Manage people in a task list**

1. **<u>Select the task list:</u>** Right-click or click the three-dots icon and select “**Manage people**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-4.57.42-PM.png" width="670" height="753" alt="" />

2. **<u>View existing subscribers</u>**<u>:</u> In the subscribers’ option, you’ll see a list of existing subscribers who have access to this task list.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl4.png" width="718" height="285" alt="" />

3. **<u>Add new subscribers</u>**<u>:</u> Click on the “**+**” icon to add new people. From the drop-down list select new people.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl5.png" width="719" height="287" alt="" />

4. **<u>Remove subscribers</u>**<u>:</u> Hover over the existing people’s names and select the “**x**” to remove them.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl6.png" width="713" height="370" alt="" />

5. **<u>Subscriber stages:</u>** Update subscriber stages if needed and click on the “**Update**” button to save and apply the changes.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl7.png" width="714" height="369" alt="" />

## **Edit a task list**

1. **<u>Edit task list:</u>** Right-click or click the three-dots icon and select “**Edit**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-4.59.28-PM.png" width="669" height="752" alt="" />

2. **<u>Edit the details:</u>** Modify the task list title, change workflow, add or remove new subscribers, and subscriber stages, etc, and click on “**Update**” to make the changes.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl9.png" width="716" height="529" alt="" />

---

## **Delete a task list**

1. **<u>Select the task list</u>**<u>:</u> Right-click or click the three-dots icon and select “**Delete**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.00.54-PM.png" width="673" height="754" alt="" />

2. **<u>Confirm deletion</u>**<u>:</u> Select “**Yes**” within this confirmation window to delete.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl-2.png" width="498" height="316" alt="" />

   \
   **<u>Note</u>****:** *Deleting this tasklist will also delete its tasks. If you want to keep the tasks, select the option “Move tasks to another tasklist”.* *ProofHub keeps items in the “Trash” for 15 days. Restore them easily by going to Trash. For detailed instructions on restoring deleted items from the trash, please refer to&#xA0;*<a href="https://help.proofhub.com/swift/manage/how-to-manage-trash" target="_blank" rel="noopener noreferrer">*Trash*</a>*.*

## **Create a copy of a task list**

1. **<u>Select the task list</u>**<u>:</u> Right-click or click the three-dots icon and select “**Create a copy**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.02.08-PM.png" width="672" height="751" alt="" />

2. **<u>Create a copy:</u>** A pop-up window will prompt you to rename the task list title, select the project, and let you copy task stages, subscribers/assignees, dates, and comments from the original task list.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl-6.png" width="718" height="449" alt="" />

---

## **Move a task list**

1. **<u>Select the task list</u>**<u>:</u> Right-click or click the three-dots icon and select “**Move**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.03.33-PM.png" width="670" height="752" alt="" />

2. **<u>Move the tasklist:</u>** A pop-up window will prompt you to rename the task list title, select the project, and let you move task stages, subscribers/assignees, dates, and comments from the task list to another project.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/tl-9.png" width="713" height="481" alt="" />

---

## **Bookmark a task list**

1. Right-click or click the three-dots icon and select “**Bookmark**“.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-04-at-5.05.15-PM.png" width="675" height="754" alt="" />

## **Search task list**

1. **<u>Access the search option:</u>** Click on the search icon within the tasks section to reveal the search bar and start typing the name of the task list you wish to search. As you type, the search results will dynamically update to display matching task lists based on the entered keywords.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-9.48.00-PM.png" width="701" height="752" alt="Access the search option in ProofHub" />

   \


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-9.50.08-PM.png" width="686" height="752" alt="Access the search option" />

2. **<u>Review search results:</u>** Browse through the search results to identify the specific task list you are looking for.
