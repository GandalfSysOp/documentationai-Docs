## **Upload a file**

1. **<u>Navigate to the files section:</u>** Click on the **“Add”** button and select **“Upload files”** from the drop-down.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Upload-a-file.png" width="1290" height="387" alt="Upload a file in ProofHub's files section" />



2. You have the option to either drag and drop the file, or you can attach files from third-party storage applications by clicking the drop-down icon adjacent to the browse button.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screen-Shot-2025-12-24-at-10.28.23.png" width="722" height="411" alt="file upload" />



3. Once you upload a file, you can select people you want to notify, choose a folder in which you want to save the file, and then click **“Ok”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/71.png" width="704" height="448" alt="" />

**<u>Note:</u>****&#xA0;**

- *The files you upload from third-party storage applications like* Dropbox, Box, and Onedriv&#x65;*&#x20;are not physically moved to your ProofHub account* *so does not count against ProofHub storage space.*

- *You are allowed to upload a file of up to 1 GB in size.*

- *You can upload multiple files at once.*

## **Move a file**

1. Right-click on the file or click on the three dots and select the **“Move”** option.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Move-a-file.png" width="1369" height="458" alt="Move a file in ProofHub's files section" />

2. Choose the folder or subfolder where you wish to move the file and click on **“Update”**.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Files-4.png" width="491" height="219" alt="" />

**<u>Note:</u>** *You can select multiple files using the&#x20;****“Ctrl/ Cmd +A”****&#x20;key. Once selected, you will see the&#xA0;****“Move”****&#xA0;option at the top. Click on it to move the files to another folder.*

## **Delete a file**

1. Right-click on the file or click on the three dots and select the **“Delete”&#xA0;**&#x6F;ption.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Delete-a-file.png" width="1356" height="457" alt="Delete a file in ProofHub's files section" />



2. Select “**Yes**” within this confirmation window to proceed with deleting a file.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Files-6.png" width="384" height="177" alt="" />

## **Sort and filter files**

In ProofHub, managing files efficiently is essential for streamlined collaboration and project organization. The sort and filter options empower users to quickly locate, organize, and access files based on specific criteria, enhancing productivity and workflow management. \
\


<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Sort-and-filter-files.png" width="1300" height="431" alt="Sort and filter files in ProofHub" />

1. **View:** By default, the **“All”&#xA0;**&#x6F;ption is selected. You can select the **“Approved”&#xA0;**&#x6F;r **“Proofed”&#xA0;**&#x6F;ption to see only those files that are either approved or proofed.

2. **Sort by:** By default, all your files are sorted based on the **“Date”** (newest to oldest). You can select the **“Date”, “Name”, “Type”, or “Size”** options to sort files as per your requirements.

3. **Order by:** You can choose between “Desc” (descending) or “Asc” (ascending) to arrange the files based on their latest updated dates.

## **View the properties of a file**

From properties, you can view file information such as type, size, upload section, upload dates, and uploaded-by details.

1. Go to the top right of the screen click on the **“i”&#xA0;**&#x69;con and select the **“Properties”** option.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Properties-of-file.png" width="1307" height="431" alt="Properties of file in ProofHub" />



2. You also have the option to view properties by either right-clicking on the file or clicking on the three dots and selecting the **“Properties”** option.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Properties-of-file-from-3-dots.png" width="1358" height="492" alt="Properties of file from 3 dots in ProofHub" />
