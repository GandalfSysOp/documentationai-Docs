title: How to import and export tasks
---
## **Import tasks**

1. **<u>Initiate task import:</u>** Select the task list where you want to import the tasks from the CSV file. Click on the “**Add**” button at the top and select “**Import CSV**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-9.55.36-PM.png" width="1440" height="755" alt="Import tasks in ProofHub" />

2. **<u>Prepare CSV file:</u>** In the import tasks window, you have the option to download a sample CSV file. Use this sample file to enter task details such as title, description, start date, due date, assignees, etc.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-16-at-12.35.59-PM.png" width="716" height="602" alt="" />

3. **<u>Upload CSV file:</u>** Browse and select or drag the CSV file into the import tasks window. From the “**Tasklist**” option in the import tasks window, choose the task list where you want to import the tasks from the CSV file and select “**Next**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-16-at-12.37.15-PM.png" width="718" height="600" alt="" />

4. **<u>Map column headers:</u>** Map the column headers from the dropdown menus according to the columns in your CSV file. The Title field is mandatory for task import. Select the date format used in your CSV file.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-16-at-12.40.37-PM.png" width="716" height="601" alt="" />

5. **<u>Map users:</u>** Map users present in your ProofHub account with the users listed in the imported CSV file. This step ensures tasks are assigned to the correct team members. Click the “**Start import**” button to initiate the task import process.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-05-16-at-12.42.17-PM.png" width="718" height="598" alt="" />


   **<u>Note:</u>**&#x20;If any users from the CSV file are not mapped to ProofHub users, imported tasks will remain unassigned. All imported tasks will be assigned to the first stage of the associated workflow.*

6. **<u>Monitor import progress:</u>** Depending on the number of tasks being imported, it may take a few minutes to complete the import process. Monitor the progress until the import is finished.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-9.58.48-PM.png" width="356" height="751" alt="Import progress in ProofHub" />

7. **<u>View import results:</u>** Once the import is completed, a success message pop-up will appear in the bottom-left corner of the screen.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-10.00.32-PM.png" width="356" height="753" alt="Successfull import progress in ProofHub" />

   Any records that were not successfully imported can be downloaded for review.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-10.10.36-PM.png" width="358" height="753" alt="Import CSV error" />

**<u>Note:</u>** *The import limit is set to 100 tasks.*

---

## **Export tasks**

1. **<u>Navigate to task lists:</u>** Start by accessing the tasks section within your project in ProofHub.

2. **<u>Download CSV:</u>** Once you’ve selected the task list, go to the top right corner of the task list and click on the three dots and select the option “**Download CSV**“.


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Screenshot-2024-06-24-at-10.12.43-PM.png" width="1440" height="752" alt="Export tasks in ProofHub" />
