---
title: How to switch between list view and thumb view
---
ProofHub offers two options for viewing your files: List view and Thumb view.

1. **List view:** It presents files in a list format, providing details such as file names, size, comments, and the option to **“Proof this file”.** The list view can be particularly helpful for users who prefer a systematic approach to file management or need to quickly scan through file details.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/List-view.png" width="1305" height="423" alt="List view in ProofHub's files section" />



2. **Thumb view:&#xA0;**&#x49;t provides users with a visual way to navigate through their files by displaying thumbnail previews. This can make it easier to quickly identify and locate specific files based on their visual content.\


   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Thumb-view.png" width="1313" height="377" alt="Thumb view in ProofHub's Files section" />
