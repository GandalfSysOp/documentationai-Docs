## **Add a project**

1. **<u>Access the “Quick add menu”</u>:** Click on the **“+”** icon located in the top left side to open the quick add menu and select **“Project”**.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/8-1.png" width="439" height="505" alt="" />

**<u>Enter the project details</u>:**

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/9-1.png" width="698" height="637" alt="" />



- Project from a template: Check the **“Use template”** box to create a project using a template. It saves time, especially for projects with similar structures. For more on templates <a href="https://hc.proofhub.in/what-is-a-template" target="_blank" rel="noopener noreferrer">click here</a>.

- **Color:&#xA0;**&#x42;y setting a color for your project, you can utilize the color filter to conveniently sort and organize your projects based on color categories.

- Add th&#x65;**&#xA0;title&#xA0;**&#x61;nd th&#x65;**&#xA0;description.**

- **Assignees:&#xA0;**&#x50;eople who should be part of the project.

- **Project manager:&#xA0;**&#x53;elect the project manager from the list of assignees assigned to the project. The project Manager is responsible for coordinating all aspects of a project, including planning, execution, and team collaboration, to ensure successful completion within defined timelines and objectives.

- Select the **start date** and **end date** of the project.

- **Project category:&#xA0;**&#x43;ategories can be used to group multiple projects under one heading, making it simpler to search for projects and generate reports by filtering projects based on their category.

- **Project status:&#xA0;**&#x49;t refers to the current stage or condition of a project, such as active, completed, on hold, archived, or any other custom status defined by you to reflect the project’s progress and status in its life cycle.

- Click **“Next”** to save the project details.

**<u>Select the project tabs</u>:**

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/10-1.png" width="705" height="562" alt="" />



- **Enable/disable the tabs:** Project tabs refer to the different sections or areas within a project. You have the option to make specific tabs visible or hidden based on your preferences.

- **Set a tab as default:** Setting a tab as default means that when you open a project, it will automatically navigate to the tab you have selected as the default. If no other tab is chosen, the **“Tasks”** tab will be set as the default.

1. **<u>Save the project</u>**: After entering all the details and selecting the tabs required for the project, click the **“Add”** button to save your project.

## **Edit a project**

1. **<u>Hover over the project name:</u>** The gear icon will be visible, click on it and select **“Edit”** from the context menu.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Project-3.png" width="633" height="603" alt="" />

2. **<u>Edit project details</u>**: In the project settings menu, you’ll find options to edit various details about the project. This may include the project title, description, category, status, and other project-specific settings.

3. Click “**Update**” to save the changes.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/22.png" width="707" height="637" alt="" />

## **Manage people on a project**

1. **<u>Hover over the project name:</u>** The gear icon will be visible, click on it, and select **“Manage people”** from the context menu.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Project-2.png" width="630" height="611" alt="" />

**<u>Remove or add people</u>**:

<Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/24.png" width="707" height="290" alt="" />

- You can remove the existing person by clicking on the **“x”** next to their name.

- You can remove all the people from the project by clicking on **“clear”**.

- You can add a new person to the existing project by clicking on the **“+”** icon.

- You can add or invite people to the ProofHub account directly from the project by clicking on the **“Add Person”** icon.

- **<u>Note:</u>** Select the project manager from the list of assignees assigned to the project.

For more on managing people in your account, <a href="https://hc.proofhub.in/how-to-add-and-manage-group" target="_blank" rel="noopener noreferrer">click here</a>.

## **Archive a project**

1. **<u>Hover over the project name:</u>** The gear icon will be visible, click on it, and select **“Archive”** from the context menu.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Project-4.png" width="633" height="621" alt="" />

2. Select “**Yes**” within this confirmation window to proceed with archiving the project.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Archive-project.png" width="584" height="217" alt="" />

**<u>Note:</u>** You can unarchive the projects from the project section on the left side menu using the **“Status”** filters at the top.

## **Delete a project**

1. **<u>Hover over the project name:</u>** The gear icon will be visible, click on it and select **“Delete”** from the context menu.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Project-5.png" width="637" height="616" alt="" />

2. Select “**Yes**” within this confirmation window to proceed with deleting the project.

   <Image src="https://help.proofhub.com/swift/wp-content/uploads/2024/05/Delete-project.png" width="586" height="216" alt="" />

**<u>Note:</u>** For detailed instructions on restoring deleted items from the trash, please refer to&#x20;*<a href="https://hc.proofhub.in/how-to-manage-trash" target="_blank" rel="noopener noreferrer">*Trash*</a>.

## **Add and manage project categories**

Project categories in ProofHub help you organize and group projects effectively, making it easier to manage and locate related projects. You can assign categories or create new categories while creating projects. **“Uncategorized”** is the default category.
