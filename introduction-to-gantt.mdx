---
title: Introduction to gantt
---
Gantt in ProofHub is an intuitive way to plan projects and visualize their progress in real time. They are easy to interpret and make an ideal way to communicate the latest project updates across team members and clients. With gantt, you can:

1. **<u>Create project schedules:</u>** You can create detailed project schedules by adding tasks directly onto the gantt. Create a task, then specify its start and end dates. You can also add task descriptions, assignees, and other relevant details.

2. **<u>Track task progress:</u>** This allows you to see which tasks are on track, which ones are behind schedule, and where adjustments may be needed to keep the project on course.

3. **<u>Set dependencies:</u>** You can easily set dependencies between tasks by linking them on the gantt. This visual representation helps you identify the sequence of tasks and understand how changes to one task may impact others.

4. **<u>View critical path:</u>** It allows you to see the tasks that are currently affecting your project’s end date. 

5. **<u>View workload allocated to resources:</u>****&#x20;**&#x59;ou can view the workload allocated to each resource directly on the gantt. This visibility helps you identify potential overloads or underutilized resources and make adjustments as needed to optimize resource allocation.
