## Latest Updates

You can track every improvement to ProofHub right here. Each release focuses on making teamwork smoother, adding useful features, and fixing issues that matter to your projects.

<Update label="2024-10-15" description="v5.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration in task boards so you can see teammate cursors and edits instantly.
- Introduced custom workflow templates that you can save and reuse across multiple projects.

## Improvements

- Enhanced dashboard loading speed by 40 percent for teams managing over 500 tasks.
- Updated the mobile app with offline mode support for task creation and comments.

## Bug Fixes

- Resolved an issue where file uploads failed for files larger than 50 MB.
- Fixed calendar sync errors that occurred with Google Workspace accounts.

</Update>

<Update label="2024-09-20" description="v5.1.3" tags={["improvement", "bugfix"]}>

## Improvements

- Refined notification settings so you receive fewer irrelevant alerts while staying informed on critical updates.
- Improved search performance across large comment threads and document libraries.

## Bug Fixes

- Corrected a display glitch in Gantt charts when viewing projects on high-resolution screens.
- Fixed export formatting for CSV reports containing special characters.

</Update>

<Update label="2024-08-12" description="v5.1.0" tags={["feature", "breaking"]}>

## New Features

- Launched AI-assisted task prioritization that suggests due dates based on your team's past performance.
- Added granular permission controls for guest users so you can limit access to specific files only.

## Breaking Changes

- Updated API endpoints for task creation. Review the migration guide at `https://docs.proofhub.com/api` before upgrading.

</Update>
